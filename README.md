# JAVA SNACKS

## Snack 1
Chiedi all’utente di continuare ad inserire un numero pari compreso tra 10 e 20. <br>
Se il numero non corrisponde alla richiesta continuare a chiederlo fino a che non viene inserito un numero valido.
